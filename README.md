# XTRA Header — Corporate Landing Page (HTML/CSS/JS)

A responsive corporate landing page template with a top bar + navbar, hero image slider with typewriter headlines, About/Services/FAQ/Contact sections, scroll animations, and a back-to-top button. Built using clean **HTML, CSS, and vanilla JavaScript** (plus Font Awesome icons via CDN).

---

## ✨ Features

- **Top bar + navigation**
- **Hero slider** with left/right arrows
- **Typewriter effect** on hero headings (driven by `data-text=""`)
- **About section** with scroll-in animations
- **Services** section with icon cards (Font Awesome)
- **FAQ** using native `<details>` / `<summary>`
- **Contact form UI**
- **Scroll animations** (left/right/zoom reveal)
- **Back-to-top** floating button
